Comandos github:
Para saber la verion: git --version
Para iniciar git: git init		 
Ver el estado del repocitorio: git status
Para añadir todos los archivos: git add .
Para hacer un commit: git commit -m "Mensaje del commit"
para actualizar siempre git add . y luego haces un commit

Imprimir todos los commits: git log --oneline (El head te dice donde estás)

Para añadir la rama git branch -M main  (con el -M cambias el nombre a la terminal)

Para ver el nombre de la rama: git branch

Para subir a la nube: git push -u origin main (solo la primera vez utilizas el -u origin main, de ahí solo basta git push)
